Задание: Учет товаров с использованием интерфейсов и абстрактных классов

Цель задания: Создать систему учета товаров с использованием интерфейсов и абстрактных классов для общей логики и специфичных методов для различных типов товаров.

Описание:

Создайте интерфейс IProduct, который будет представлять общий функционал для всех типов товаров. В интерфейсе должны быть объявлены следующие методы:

void AddToInventory(int quantity): Метод, который будет добавлять товары на склад с указанным количеством.
void RemoveFromInventory(int quantity): Метод, который будет удалять товары со склада с указанным количеством.
void GetProductInfo(): Метод, который будет выводить информацию о товаре.
Создайте абстрактный класс ProductBase, который будет реализовывать интерфейс IProduct. В этом абстрактном классе определите общую логику для всех типов товаров. Например, методы AddToInventory и RemoveFromInventory могут иметь общую реализацию для всех товаров, а метод GetProductInfo может выводить общую информацию о товаре.

Создайте два класса, которые наследуются от абстрактного класса ProductBase:
ElectronicsProduct: Класс, представляющий электронику. Переопределите метод GetProductInfo для вывода информации о конкретных характеристиках электроники, таких как модель, производитель и т.д.
ClothingProduct: Класс, представляющий одежду. Переопределите метод GetProductInfo для вывода информации о конкретных характеристиках одежды, таких как размер, цвет и т.д.
В основной части программы создайте объекты ElectronicsProduct и ClothingProduct и продемонстрируйте использование метода AddToInventory, RemoveFromInventory и GetProductInfo для каждого из классов.


Ответьте на следующие вопросы:
В чем основное различие между интерфейсами и абстрактными классами в этой задаче?
Когда следует использовать интерфейсы, а когда абстрактные классы в разработке системы учета товаров? Какие преимущества и недостатки каждого подхода?
Какие другие типы товаров могут быть добавлены в программу, и какие изменения нужно будет внести в интерфейс и классы для их поддержки?

Задание: Управление строительством с использованием интерфейсов и абстрактных классов

Цель задания: Создать систему управления строительством с использованием интерфейсов и абстрактных классов для общей логики и специфичных методов для различных типов строительных объектов.

Описание:
Создайте интерфейс IBuilding, который будет представлять общий функционал для всех типов строительных объектов. В интерфейсе должны быть объявлены следующие методы:

void StartConstruction(): Метод, который будет начинать строительство объекта.
void CompleteConstruction(): Метод, который будет завершать строительство объекта.
void CalculateCost(): Метод, который будет вычислять стоимость строительства объекта.
Создайте абстрактный класс BuildingBase, который будет реализовывать интерфейс IBuilding. В этом абстрактном классе определите общую логику для всех типов строительных объектов. Например, методы StartConstruction и CompleteConstruction могут иметь общую реализацию для всех объектов, а метод CalculateCost может включать общую формулу для расчета стоимости строительства.

Создайте два класса, которые наследуются от абстрактного класса BuildingBase:
HouseBuilding: Класс, представляющий строительство дома. Переопределите методы StartConstruction, CompleteConstruction и CalculateCost для соответствующей реализации строительства дома.
OfficeBuilding: Класс, представляющий строительство офисного здания. Переопределите методы StartConstruction, CompleteConstruction и CalculateCost для соответствующей реализации строительства офисного здания.
В основной части программы создайте объекты HouseBuilding и OfficeBuilding и продемонстрируйте использование методов StartConstruction, CompleteConstruction и CalculateCost для каждого из классов.

Ответьте на следующие вопросы:
В чем основное различие между интерфейсами и абстрактными классами в этой задаче?
Когда следует использовать интерфейсы, а когда абстрактные классы в управлении строительством? Какие преимущества и недостатки каждого подхода?
Какие другие типы строительных объектов могут быть добавлены в программу, и какие изменения нужно будет внести в интерфейс и классы для их поддержки?

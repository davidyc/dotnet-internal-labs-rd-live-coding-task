Задание на работу со ссылочными и значимыми типами, а также понимание разницы между ref и out:

Создайте класс DataProcessor, в котором будут следующие методы:

ModifyReferenceType: Метод для изменения значения ссылочного типа (например, класса). Подумайте, как работает передача ссылочного типа по значению.
ModifyValueTypeWithRef: Метод для изменения значения значимого типа (например, целого числа) с помощью ключевого слова ref. Подумайте, как работает передача значимого типа по ссылке.
ModifyValueTypeWithOut: Метод для изменения значения значимого типа (например, числа с плавающей точкой) с помощью ключевого слова out. Понимание работы out очень важно для корректной инициализации переменных перед их использованием.
Создайте класс Student для представления студента (это будет использоваться для демонстрации работы с ссылочным типом).

В методе Main:

Создайте объект класса DataProcessor.
Создайте несколько объектов класса Student и переменных значимых типов.
Вызовите методы класса DataProcessor, передавая в них созданные объекты и переменные с различными значениями.
Обратите внимание на разницу в поведении ref и out, а также на то, как они влияют на значения переданных аргументов.
Выведите результаты после вызова методов.
Цель задания: Данное задание позволит студентам усвоить различия между ссылочными и значимыми типами данных, а также научиться использовать ключевые слова ref и out для передачи значимых типов в методы. Понимание разницы между ref и out позволит студентам корректно использовать эти ключевые слова в различных ситуациях при работе с параметрами методов.

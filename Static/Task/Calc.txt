Задание на тему "Статические классы, переменные и методы":

Цель задания: Понять разницу между статическими и экземплярными классами, переменными и методами.

Создайте класс Calculator, который будет иметь следующие свойства:

Статическое целочисленное поле result.
Статический метод Add(int a, int b) для сложения двух чисел и сохранения результата в поле result.
Статический метод Subtract(int a, int b) для вычитания второго числа из первого и сохранения результата в поле result.
Статический метод GetResult() для получения значения текущего результата из поля result.
Создайте класс Counter, который будет иметь следующие свойства:

Статическое целочисленное поле count.
Экземплярный метод Increment() для увеличения значения поля count на 1.
Экземплярный метод Decrement() для уменьшения значения поля count на 1.
Экземплярный метод GetCount() для получения значения текущего счетчика.
В методе Main:

Создайте объекты классов Calculator и Counter.
Вызовите статические методы Add и Subtract класса Calculator с различными значениями и выведите результаты.
Вызовите экземплярные методы Increment и Decrement класса Counter и выведите значения счетчика.
Обратите внимание на различие в использовании статических и экземплярных методов, а также на то, как значения статических переменных сохраняются между вызовами методов.
